## Chạy các ví dụ

* Chạy `npm i`
* Di chuyển đến thư mục ví dụ
* Chạy `npm run build` (hoặc `npm run watch` nếu bạn muốn theo dõi các thay đổi)
* Mở file trong thư mục `public` bằng trình duyệt
